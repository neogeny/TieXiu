For error reporting you will create a Memento struct that saves enough information to implement Display and Debug properties that print error messages much like compiler like rustc or python. The inputs to construction ar text: &str, start: usize, and mark: usize, which represent the complete input, the start of an error, and the last position of interest. The key feature of the information saved for rendering is a windows of (usize, Box<str>) with the absolute line number and the text of the lines relevant to the error. In this version of Memento the window will contain the 4 lines before the mark line. In this version of Memento the window will contain continuos lines but in the future it may contain disparate sets. Part of the saved information is the relative line numbers and line positions for the start and mark arguments given for construction. The Display and Debug trait implementations will produce output similar to what the Rust compiler produces using the console crate for colorization. The implementation will be a single rust module.
